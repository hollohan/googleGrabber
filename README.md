#googleGrabber

This code is a class that can be used to get the URLs on a google search.

###Requirements
    - selenium (http://www.seleniumhq.org/)


###Usage
    >googleGrabber.Grabber(
        >rootURL='https://www.google.com/webhp?hl=en#hl=en&',
        >searchPhrase='google',
        >filterKiller='&filter=0',
        >filename=''
    >)
